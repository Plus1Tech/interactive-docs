One-Time Setup
Before You Begin
3 Steps to Set Up
Your Claude Project
Complete these steps once to activate the HOCK Content Production Agent. All three must be done before running any phase prompts.
1
Paste the System Prompt into Project Instructions
Navigate to: claude.ai → Your Project → Edit project → Project Instructions
  • 📋
    Find the system prompt in your Monday.com board → AI Prompt Library → Claude Project folder. All prompts you'll ever need for this agent live there.
  • 📄
    Open HOCK_System_Prompt_3_24_26_Final.md from the AI Prompt Library and copy the entire contents.
  • ✏️
    Paste the full contents into the Project Instructions text field inside your Claude project settings.
  • 🚫
    Do NOT upload the system prompt as a file to Project Knowledge. It must go in the text field only — phase-locked behavior won't activate otherwise.
  • 💡
    The text field is inside Edit project → Project Instructions, not the Knowledge tab. These are two different places.
2
